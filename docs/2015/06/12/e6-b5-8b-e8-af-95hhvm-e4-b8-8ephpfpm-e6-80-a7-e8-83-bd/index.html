测试hhvm 与phpfpm性能

Posted on Jun 12, 2015 By Ebankie

总是说hhvm性能很高亲测试数据如下：

从cpu\mem\io 三个方面

服务器：

cpu 8核 3.30GHz

mem 16G

disk 500G

一、time 测试：

网上的例子CUP

test.php

<?php
function fib($n)
{
    if ($n <= 2)
        return 1;
    else
        return fib($n-1) + fib($n-2);
}

$n = 36;
printf("fib(%d) = %d\n", $n, fib($n, 2));
?>
root#time hhvm test.php

real 0m0.407s user 0m0.336s sys 0m0.068s

root #time php test.php

real 0m1.046s user 0m0.952s sys 0m0.092s

内存及IO 也相差类似
